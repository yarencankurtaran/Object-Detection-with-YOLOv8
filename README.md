# Object-Detection-with-YOLOv8
In this project, a YOLOv8-based object detection model was trained using the MiDeSec dataset. Model performance was analyzed by comparing raw images with pre-processed and data-enhanced images. 

## Dataset
The dataset is available on Kaggle:
https://www.kaggle.com/datasets/sonianmty/nusec-and-midesec

## How to run
1. Download dataset from Kaggle
2. Put into /data folder
3. Run notebook
